# Notification Sound File

Place a notification sound file named `notification.mp3` in this directory.

## Requirements:
- File name: `notification.mp3`
- Format: MP3 audio file
- Duration: 1-3 seconds recommended
- Volume: Moderate level (not too loud)

## Suggested Sound Types:
- Pleasant chime or bell sound
- Soft notification tone
- Brief musical note

## Usage:
This sound file is played when a new order is received in the admin dashboard to alert staff members.

## Note:
The file path `/sounds/notification.mp3` is referenced in:
- `src/components/admin/orders/NewOrderNotification.tsx`

Make sure to add an appropriate audio file here for the notification system to work properly.
